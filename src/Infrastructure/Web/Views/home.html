Task Clean Architecture

By Anglesson Araujo

This project was created to show how to implements The Clean Architecture in practice. When I started this project, it was a simple example to explain to my friend Jefferson how to start a project using clean architecture. I was excited by everything that I read in the book. Then, I decided to apply the maximum number of concepts that I could. Then the Task Clean Architecture project was born, simple to understand and follow.

About the project

The Task Clean Architecture project has 5 functionalities, all them covered by tests using PHPUnit:


Functionalities:
  1. Create
  2. Find
  3. Update
  4. Delete
  5. List
Stack
  1. Language: PHP
  2. Framework of tests: PHPUnit
  3. Slim Framework in external layer
